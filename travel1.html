来日前（Before Arrival）

準備を確認する ↓
01

ビザ確認

日本へ旅行する前に、自分の国籍でビザが必要か確認します。

いつやる？

航空券を買う前、できれば旅行予定日の1〜2か月前に確認します。

準備するもの

  • パスポート
  • 国籍情報
  • 旅行目的

やり方

  1. 外務省または日本大使館のWebサイトを開く
  2. 自分の国籍がビザ免除対象か確認する
  3. ビザが必要な場合、必要書類を確認する
  4. 日本大使館・領事館、またはeVISAで申請できるか確認する
  5. ビザが発給されてから航空券・ホテルを最終確認する

担当：本人・日本大使館

02

パスポート

有効期限と名前のスペルを確認します。

いつやる？

旅行計画を立てる前に確認します。

必要なもの

  • パスポート
  • パスポートコピー
  • スマホに保存したパスポート写真

やり方

  1. パスポートの有効期限を確認する
  2. 名前のスペルが航空券と同じか確認する
  3. 顔写真ページをコピーまたはスマホに保存する
  4. 旅行中は原本とコピーを別々に保管する

担当：本人

03

航空券・ホテル

日本での滞在先住所は、ビザ申請、Visit Japan Web、入国審査、ホテルチェックインで使います。 ホテル名、住所、電話番号をメモしておくと安心です。

必要なもの

  • 航空券予約確認書（航空会社）
  • ホテル予約確認書（ホテル）
  • ホテル住所（ホテル）
  • ホテル電話番号（ホテル）
  • 滞在日数（本人）

やり方

  1. 入国日と帰国日を決める
  2. 往復航空券を予約する
  3. ホテルまたは宿泊先を予約する
  4. 予約確認メールをPDFまたはスクリーンショットで保存する
  5. ホテル住所を英語・日本語でメモしておく

担当：本人・航空会社・ホテル

04

海外旅行保険

旅行予定表は、ビザ申請や入国審査で確認されることがあります。 いつ、どこに泊まり、どこへ行くかを簡単にまとめます。

必要なもの

  • 旅行日程表（本人）
  • ホテル予約確認書（ホテル）
  • 航空券情報（航空会社）
  • 観光予定地リスト（本人）

やり方

  1. 1日目から帰国日までの日程を書く
  2. 宿泊ホテルを書く
  3. 訪問予定地を書く
  4. 空港からホテルまでの移動方法を書く
  5. PDFまたは紙で持っておく

担当：本人

外務省：申請書類ダウンロード
05

お金・支払い方法の準備

日本ではクレジットカードが使える場所も多いですが、小さい店、地方、交通機関、自動販売機などでは現金が必要な場合があります。

必要なもの

  • 日本円の現金（本人）
  • クレジットカード（本人・カード会社）
  • デビットカード（本人・銀行）
  • 海外で使えるカードか確認（銀行・カード会社）

やり方

  1. 少し日本円を準備する
  2. カードが海外利用可能か確認する
  3. ATMで引き出せるか確認する
  4. カードをなくした時の連絡先をメモする
  5. 現金とカードを別々に保管する

担当：本人・銀行・カード会社

06

Visit Japan Web 登録

いつやる？

日本へ出発する前に登録しておくと便利です。

必要なもの

  • パスポート（本人）
  • メールアドレス（本人）
  • 航空便情報（航空会社）
  • 日本での滞在先住所（ホテル）
  • 税関申告情報（本人）
  • スマートフォン（本人）

やり方

  1. Visit Japan Webにアクセスする
  2. アカウントを作成する
  3. 本人情報を登録する
  4. パスポート情報を登録する
  5. 入国予定日と航空便情報を入力する
  6. ホテル住所を入力する
  7. 税関申告情報を入力する
  8. QRコードを保存する
  9. 空港でQRコードを提示する

担当：本人・入管・税関

07

海外旅行保険

万が一に備えて旅行保険へ加入します。

必要なもの

  • 保険証書（保険会社）
  • 保険番号（保険会社）
  • 緊急連絡先（保険会社）
  • 病院で使える言語サポート情報（保険会社）

やり方

  1. 旅行保険に入るか確認する
  2. 病気・けが・盗難・飛行機遅延が対象か確認する
  3. 保険会社の緊急連絡先を保存する
  4. 保険証書をスマホと紙の両方で持つ

担当：本人・保険会社
